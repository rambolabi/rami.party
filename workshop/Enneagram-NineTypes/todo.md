# The Enneagram — Workshop Project

## Overview
Interactive workshop on the nine Enneagram types (motivation-based personality).
Part of **The People Library** (../personality-library/index.html).

## Delivered
- Shared engine (index.html / style.css / app.js), pink accent.
- `data.js` — content, `classify` assessment (9 questions → one of nine types), 9 type profiles.
- Concept cards (wings, arrows, growth/stress), apply guide, FAQ, DISC colour cross-link.
- Responsive, accessible, print-friendly. No backend.

## Notes
- Balanced triad rotation so each of the nine types is covered evenly.
- Framed as a reflection tool, not a scientific instrument.
