    Help & shortcuts

    Laser Forge turns text, shapes, images, QR/barcodes, puzzles and parametric parts into machine-ready files — 100% in your browser.

    • 1) Add an object from the left toolbar.
    • 2) Tune it in the Properties panel (turn on Pro mode for advanced fields).
    • 3) Press Export and pick your laser software preset.

    The tools (left bar) — when to use what

    • 🅣 Text — words/labels. Any font; “Arc” bends it. Exported as outlines so the font isn’t needed on the machine.
    • ☺ Emoji — drop a symbol; it’s traced to a clean engravable silhouette.
    • 🖼 Image — a photo/logo for grayscale engraving. Use the sliders + dithering, then export PNG/BMP/JPG or G-code.
    • ▦ QR / ▮▮ Barcode — scannable codes (URL, Wi-Fi, product IDs).
    • 🧩 Puzzle — jigsaw, tessellation, geometric, Voronoi, or Custom (draw your own repeating tile).
    • ▢ Box — finger-joint enclosure net; set W×D×H + material thickness, then cut & assemble.
    • ⚙ Gear — a gear with chosen teeth + bore. 📏 Ruler — graduated ruler; set length via Width.
    • 〜 Hinge — living hinge (kerf slits) so flat wood bends. ✚ Marks — registration crosshairs for print-and-cut.
    • Shapes — rectangle, circle, polygon, star, heart, line for building your own artwork.

    Operations (the colour of each object)

    • ● Cut — cut all the way through (outlines).
    • ● Score — light surface line (fold/mark).
    • ● Engrave — fill/darken an area (text, photos).

    Which export format?

    • SVG — best all-round vector (LightBurn, Glowforge, xTool, Inkscape, Rayforge).
    • DXF — vector for Ruida/RDWorks & CAD. PDF/AI/EPS — vector for design apps.
    • PLT — plotter/HPGL vector. PNG/BMP/JPG — raster for photo engraving (BMP = LaserGRBL).
    • G-code — runs directly on a GRBL machine. Verify first — never run a laser unattended.
    • Tip: pick your machine under Target software and it sets the format for you.

    Handy options

    • Dithering (images) — fakes grey shades with dots; best for photos on wood/acrylic.
    • Kerf comp. (export) — grows cut lines by half the beam width so parts fit exactly.
    • Overscan (raster G-code) — runs past the edges to avoid dark burn marks.
    • Array… — duplicate the selection into a grid (e.g. 3x2x5 = 3×2 with 5 mm gaps).
    • Save/Open keep your project as a file; 🔗 link shares it via URL.

    Shortcuts

    • Ctrl+Z / Ctrl+Y — undo / redo
    • Ctrl+D — duplicate · Del — delete
    • F — fit view · scroll — zoom · Space+drag — pan
    • Drag a handle to resize, the top dot to rotate (hold Shift to snap).
    • Drag & drop an image straight onto the canvas.
