# The 16 Types (MBTI-style) — Workshop Project

## Overview
Interactive workshop on the four-axis, 16-type model (Myers–Briggs / Jung), presented with honest
scientific caveats. Part of **The People Library** (../personality-library/index.html).

## Delivered
- Shared engine (index.html / style.css / app.js), purple accent.
- `data.js` — content, `axes` assessment (12 either/or → 4-letter type), all 16 type profiles.
- Concept cards, apply guide, FAQ, DISC colour cross-link.
- Responsive, accessible, print-friendly. No backend.

## Notes
- Assessment mode `axes`: tallies four dichotomies (E/I, S/N, T/F, J/P) into a 4-letter type.
- Deliberately caveated: results can shift; use as a mirror, pair with the Big Five.
