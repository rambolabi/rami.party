# Surrounded by Liars — Web Application Project

## Project Overview
An educational, interactive workshop based on Thomas Erikson's *Surrounded by Liars*. It explains
how deception really works and tests the visitor's lie-detection literacy with a myth-or-fact quiz.

## Key Requirements
- Educational content: why we lie, the anatomy/escalation of a lie, myths of lie detection,
  what actually helps, and building a truth culture.
- Knowledge assessment (quiz): 10 myth-or-fact questions → score, band, and a per-question
  review with explanations.
- Handle / field guide: Do / Don't + the truth-friendly conversation.
- FAQ.

## Design & UX
- Clean, modern, professional, teal accent.
- Responsive, accessible, print-friendly, one question at a time, explanations at end.

## Technical Implementation
- Shared data-driven engine (`app.js`) + `style.css`.
- Book-specific content in `data.js` (`BOOK` object, assessment mode `quiz`).
- No backend, static site.

## Content Authority
Inspired by *Surrounded by Liars* by Thomas Erikson. Educational only — not a tool to accuse or
"prove" anyone is lying. Footer invites readers to buy the book.
