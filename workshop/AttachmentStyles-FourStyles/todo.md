# Attachment Styles — Workshop Project

## Overview
Interactive workshop on the four adult attachment styles (Bowlby / Ainsworth and later research).
Part of **The People Library** (../personality-library/index.html).

## Delivered
- Shared engine (index.html / style.css / app.js), teal accent.
- `data.js` — content, `classify` assessment (12 questions → one of four styles), 4 profiles.
- Concept cards (origins, earned security, pairing dynamics), apply guide, FAQ.
- Responsive, accessible, print-friendly. No backend.

## Notes
- Styles: secure, anxious, avoidant, fearful-avoidant.
- Framed compassionately: adaptations not flaws; styles can change ("earned secure"). Not therapy.
