# Surrounded by Setbacks — Web Application Project

## Project Overview
An educational, interactive workshop based on Thomas Erikson's *Surrounded by Setbacks*. It
explains the setback spiral and helps visitors measure and build a resilient response.

## Key Requirements
- Educational content: the setback spiral, reaction vs response, the reframe (temporary /
  specific / workable), the comeback steps, and building resilience.
- Self-assessment (score): 12 resilience statements → percentage resilience level, banded
  (Reactive / Developing / Resilient) with tailored advice and a score meter.
- Handle / field guide: Do / Don't + the comeback routine.
- FAQ.

## Design & UX
- Clean, modern, professional, blue accent.
- Responsive, accessible, print-friendly, one statement at a time, result at end.

## Technical Implementation
- Shared data-driven engine (`app.js`) + `style.css`.
- Book-specific content in `data.js` (`BOOK` object, assessment mode `score`).
- No backend, static site.

## Content Authority
Inspired by *Surrounded by Setbacks* by Thomas Erikson. Educational and reflective — not therapy
or clinical advice. Footer invites readers to buy the book and to seek professional support if
they are struggling.
