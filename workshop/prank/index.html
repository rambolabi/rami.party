Upside Down World
Easy
Rotate their screen 180 degrees for an instant headache! 
DisplaySwitch.exe /extend
Start-Sleep -Seconds 1
# Note: Screen rotation via CMD requires graphics driver support
# Ctrl+Alt+Arrow keys work on most Intel graphics
Quick method: Ctrl + Alt + ↓ to flip upside down
(Fix: Ctrl + Alt + ↑)

Left-Handed Mouse
Easy
Switch to left-handed mouse mode and watch the confusion! 
reg add "HKCU\Control Panel\Mouse" /v SwapMouseButtons /t REG_SZ /d 1 /f

Fake Desktop
Epic
Screenshot desktop, set as wallpaper, hide icons = frozen screen! 
# Take screenshot and set as wallpaper
Add-Type -AssemblyName System.Windows.Forms
$screen = [System.Windows.Forms.Screen]::PrimaryScreen.Bounds
$bitmap = New-Object System.Drawing.Bitmap($screen.Width, $screen.Height)
$graphics = [System.Drawing.Graphics]::FromImage($bitmap)
$graphics.CopyFromScreen($screen.Location, [System.Drawing.Point]::Empty, $screen.Size)
$path = "$env:USERPROFILE\Pictures\prank.png"
$bitmap.Save($path)
reg add "HKCU\Control Panel\Desktop" /v Wallpaper /t REG_SZ /d $path /f
rundll32.exe user32.dll,UpdatePerUserSystemParameters
Or: Win + Shift + S → Save → Right-click image → Set as background → Hide icons
